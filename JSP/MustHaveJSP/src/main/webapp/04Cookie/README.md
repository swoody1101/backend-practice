  <p>
    쿠키는 클라이언트의 상태 정보를 클라이언트에 저장하는 기술
  </p>
  <ul>
    <li>쿠키는 생성자를 통해서만 생성할 수 있습니다. 생성 후 쿠키값은 변경할 수 있으나 쿠키명은 변경할 수 없습니다.</li>
    <li>setPath() 메서드로 적용할 경로를 설정합니다.</li>
    <li>setMaxAge() 메서드로 쿠키의 유지 기간을 설정합니다.</li>
    <li>response 내장 객체의 addCookie() 메서드로 클라이언트에 쿠키를 저장합니다.</li>
    <li>쿠키는 생성 직후 바로 사용할 수는 없습니다. 클라이언트가 재요청을 했을 때부터 사용할 수 있습니다.</li>
  </ul>
</body>
